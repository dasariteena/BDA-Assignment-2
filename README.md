## 📖 Project Overview

This repository contains practical implementations of core Big Data Analytics concepts using **Apache Spark** on **Google Colab**.

The project demonstrates:

* Supervised Learning (Classification)
* Unsupervised Learning (Clustering)
* Recommendation Systems

All implementations are performed on real-world datasets to showcase scalability and efficiency using distributed computing.

## 🔍 Implementations

### 1️⃣ Classification Model

* Built using **Spark MLlib**
* Applied on a chosen dataset
* Steps included:

  * Data preprocessing
  * Feature engineering
  * Model training (e.g., Logistic Regression / Decision Tree)
  * Model evaluation (Accuracy, Precision, Recall)

---

### 2️⃣ Clustering Model

* Implemented using **K-Means Clustering**
* Key steps:

  * Data cleaning
  * Feature scaling
  * Cluster formation
  * Visualization of clusters

---

### 3️⃣ Recommendation Engine

* Developed using **Collaborative Filtering**
* Techniques used:

  * ALS (Alternating Least Squares)
* Capabilities:

  * Predict user preferences
  * Recommend items based on user behavior
